## 核心原则

1. **不卖医院，卖确定性** — 客户恐惧的不是价格，是在陌生国度被宰、被骗、被耽误
2. **先诊断后开药** — 搞清楚客户具体情况之前不报价
3. **$49 是入口，$399 是收入** — 先用低价产品建立信任，再升级
4. **24小时内有回复，4小时内有实质进展** — 医疗咨询的黄金窗口很短
5. **不承诺治疗结果** — 我们只承诺帮客户到达正确的医院门口


### ⚠️ 开场禁忌
- ❌ 不要说 "Sure, we can help!" 然后直接报价
- ❌ 不要问已经在网站上填过的信息
- ❌ 不要超过1小时才回复第一条消息
- ❌ 不要用语音消息（文字方便客户翻阅和转发给家人）

### 信息收集（诊断阶段）

等客户回复第一条后，根据情况问以下问题的子集：

```
为了更好地帮您匹配医院，我想了解几个关键信息：

1. 具体是什么疾病/需要什么治疗？
2. 有最近的检查报告吗？（可以拍照发我，中文/英文都行）
3. 您目前在哪个国家？
4. 对来华时间有预期吗？（尽快 / 1-3个月内 / 还在了解阶段）
5. 有没有已经在关注的医院或医生？

这些信息帮我给您更准确的建议，不着急，您方便的时候回复。
```

## 第三阶段：异议处理

### 异议1: "太贵了，我自己Google也能找到医院"

```
完全理解！Google确实能找到医院名字，但几个关键信息网上查不到：

1. 这家医院的国际患者部是否靠谱？（有些医院有国际部但根本联系不上）
2. 实际需要多少钱？（网上的价格经常是3年前的，或者不含关键费用）
3. 我这个具体病情最适合哪家？（排名不代表一切，有的医院名气大但不接国际患者）

$49的匹配报告就是帮你省掉这些摸索时间的。如果报告不解决你的问题，我退你钱。

要不然先试试$49的？觉得有用再考虑升级。
```
