# OpenClaw Anything

This skill allows execution of arbitrary OpenClaw commands.

## Usage

Use this skill when you need to run OpenClaw CLI commands directly.

## Commands

- `openclaw <command>` - Execute OpenClaw CLI commands
- `openclaw skills <subcommand>` - Manage skills
- `openclaw config <subcommand>` - Manage configuration

## Examples

```bash
openclaw skills list
openclaw config get
openclaw status
```
