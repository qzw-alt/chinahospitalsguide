# Daily ops checklist

Use this as your simple daily routine.

## Morning

1. Open `Dashboard`
2. Check:
   - `Need reply today`
   - `Paid but awaiting action`
   - `Waiting for records`
3. Open `Daily Queue`
4. Work from top to bottom:
   - high priority first
   - overdue follow-up second
   - new leads third

## After every reply

Update these fields before you move on:

- `last_contacted_at`
- `next_action`
- `next_follow_up_date`
- `lead_status`
- `follow_up_stage`

## After every payment

1. Confirm payment in PayPal
2. Fill:
   - `paypal_reference`
   - `payment_amount_usd`
3. Change:
   - `payment_status`
   - `lead_status`
   - `follow_up_stage`

## End of day

1. Filter `Daily Queue`
2. Make sure no serious case is left blank in:
   - `assigned_to`
   - `next_action`
   - `next_follow_up_date`
3. Move completed cases to `Closed`

## One rule that saves the most time

If a row has no `next_action`, it is not really being managed yet.
