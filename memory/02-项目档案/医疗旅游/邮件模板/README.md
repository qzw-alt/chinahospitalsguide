# 医疗旅游项目 - 邮件模板库

> 最后更新：2026-03-20

## 📧 模板清单

### 1. 自动回复 - 客户咨询
**文件名：** `01_自动回复_客户咨询.txt`

**使用场景：**
- 客户首次通过网站表单或邮件联系时自动发送
- 用于收集客户基本信息（病情、城市偏好、时间、国家等）

**发送时机：**
- 客户提交咨询表单后即时自动发送

**关键内容：**
- Demi自我介绍（AI助手背景）
- 4步服务流程说明
- 5个问题收集需求
- 联系方式（WhatsApp/Email）

---

### 2. 支付确认
**文件名：** `02_支付确认.txt`

**使用场景：**
- 客户完成PayPal支付后发送
- 确认订单信息并告知下一步

**发送时机：**
- 收到PayPal支付通知后立即发送

**关键内容：**
- 订单详情（ID、套餐、金额、日期）
- 24小时内交付承诺
- 套餐内容清单
- 支持联系方式
- 医疗旅程时间线

**变量说明：**
- `{{customer_name}}` - 客户姓名
- `{{order_id}}` - 订单编号
- `{{package_name}}` - 套餐名称（Starter Guide/Standard Package）
- `{{amount}}` - 支付金额（30/299）
- `{{payment_date}}` - 支付日期
- `{{#if starter_package}}` - 条件判断（根据套餐类型显示不同内容）

---

### 3. 客户攻略 - 付费版
**文件名：** `03_客户攻略_付费版.md`

**使用场景：**
- 客户支付后发送的详细医疗旅行指南
- 根据具体病情定制的完整攻略

**发送时机：**
- 支付确认邮件发送后24小时内
- 需要人工根据客户病情定制后发送

**关键内容：**
- 病情分析和医院推荐
- 签证申请指南（S2签证）
- 医院联系方式和地址
- 交通指南（机场到医院）
- 费用估算
- 住宿建议
- 治疗后跟进

**使用方法：**
1. 复制模板
2. 替换占位符：
   - `{{customer_name}}` - 客户姓名
   - `{{condition}}` - 病情（如：Chest Pain）
   - `{{order_id}}` - 订单号
   - `{{package}}` - 套餐类型
3. 根据病情选择对应医院
4. 发送给客户

## 📝 使用指南

### 发送方式

**自动邮件（推荐）：**
- 使用AgentMail API自动发送
- 脚本：`auto_reply_bot.py`
- 配置：设置触发条件和模板路径

**手动发送：**
- 复制模板内容
- 替换变量
- 通过邮件客户端发送

### 变量替换格式

**简单变量：**
```
{{variable_name}}
```

**条件判断：**
```
{{#if condition}}
显示内容
{{/if}}
```

**示例：**
```
Dear {{customer_name}},

Your order {{order_id}} has been confirmed.

{{#if standard_package}}
You will receive full coordination service.
{{/if}}
```
