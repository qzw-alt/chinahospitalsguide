## ⚙️ 融合写作规则

**核心原则：一篇两用 — 新闻热点 + 长尾指南融合**

1. 选题时优先从「内容日历」选P0-P1词
2. 搜索当日/近3天相关新闻热点
3. 如果热点恰好匹配一个待写长尾词 → 写融合文
4. 如果当天无匹配热点 → 从内容日历选一个长尾词写指南（不加新闻框架）
5. 每篇文章在发布后更新本文件「已写文章清单」
