## 🔑 核心原则

### 记忆管理
> **"记忆是AI的灵魂，没有记忆就没有智能"**

**三层架构：**
- 🔥 **HOT** - 最近15天，每日详细记录
- 🌡️ **WARM** - 15-90天，稳定配置
- ❄️ **COLD** - 90天+，历史归档

### 技能运用
> **"不是技能越多越好，而是最佳实践固化"**
> **"一山不容二虎，同类型只留最优"**

**黄金公式：**
```
Multi Search（广度）→ Tavily（深度）→ 整合输出
```

**技能迭代原则：**
- 新技能出现 → 立即对比测试 → 优胜劣汰
- 同类型技能只能保留1个最优解
- 满身是刀不如一刀致命

### 决策记录
**必须记录：** 时间、决策、原因、结果
