## 版本号规则

- **主版本号（Major）**：重大功能更新、架构变更
- **次版本号（Minor）**：新功能添加、页面增加
- **修订号（Patch）**：bug修复、小优化

## 下次更新建议

当你看到版本号变化时，说明网站已更新：
- v2.1.0 → v2.1.1：小修复
- v2.1.0 → v2.2.0：新功能
- v2.1.0 → v3.0.0：重大更新
